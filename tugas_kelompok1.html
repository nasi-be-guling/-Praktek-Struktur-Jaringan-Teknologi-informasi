TUGAS KELOMPOK

LIVE DEMO DATABASE, CRUD & JOIN QUERY

Presentasi Live & File Upload (H-1) 3 - 4 Orang per Kelompok Jadwal: Hari ini s/d menjelang UAS/Evaluasi Akhir

A. Deskripsi Tugas

Sebagai tim Database Engineer Rumah Sakit, kelompok Anda ditugaskan untuk membangun fondasi database Klinik dari nol dan mempresentasikannya secara langsung (Live Demo) di hadapan Instruktur.

Selama presentasi, Anda DILARANG KERAS menggunakan tombol antarmuka (GUI) phpMyAdmin untuk mengeksekusi operasi. Semua tindakan mutlak harus dieksekusi menggunakan Syntax SQL murni di tab SQL sembari dijelaskan kepada audiens.

B. Aturan Anti-Plagiasi & Persiapan

  • Nama Database Unik: Database harus dibuat dengan nama db_klinik_kel[NomorKelompok]_[Kelas]. (Contoh: db_klinik_kel5_2A1).
  • Data Dummy Wajib: Pada tabel pasien atau dokter, 3 data pertama WAJIB menggunakan Nama Asli dan NIM dari anggota kelompok kalian sendiri.
  • Persiapan Live Demo: Siapkan catatan (script/cheat-sheet) query SQL Anda di Notepad, agar saat presentasi Anda bisa langsung copy-paste atau mengetik dengan cepat di tab SQL phpMyAdmin tanpa membuang waktu.

C. Skenario Presentasi & Pembagian Peran

Setiap kelompok diberikan waktu 10 Menit untuk presentasi. Pekerjaan dibagi rata, di mana setiap anggota wajib mengambil alih Screen Share / Perangkat Presentasi saat gilirannya tiba:

ANGGOTA 1
The Data Architect

Fokus Struktur (DDL)

  1. Menunjukkan database yang sudah dibuat.
  2. Menunjukkan 3 tabel (pasien, dokter, kunjungan) dan menjelaskan Primary Key & Foreign Key-nya.
ANGGOTA 2
Data Entry Specialist

Fokus Manipulasi (DML)

  1. Mendemonstrasikan eksekusi 1 perintah INSERT INTO secara live.
  2. Memastikan ada data jebakan (Misal: 1 Dokter tanpa pasien) untuk bahan Anggota 4 nanti.
ANGGOTA 3
Relational Specialist

Fokus INNER JOIN

  1. Mengetik/eksekusi query SELECT ... INNER JOIN 3 tabel secara live.
  2. Membuktikan bahwa nama anggota (sebagai pasien) berhasil digabungkan di tabel hasil join.
ANGGOTA 4
Analytical Expert

Fokus LEFT JOIN & Analisis

  1. Mengeksekusi query LEFT JOIN ke tabel kunjungan.
  2. Menjelaskan kepada Instruktur: "Mengapa muncul nilai NULL pada dokter tersebut?"
